This application is a simple loan calculator REST API using Spring MVC.

Features: 1. CRUD REST API on http://localhost:8080/fundnesia/api/loans
	  2. Exception Handling


Environtment :
	1. Spring Framework 5.2.5.RELEASE
	3. Hibernate
	4. MySQL Server 5.5
	5. Tomcat 9 (Only work for tomcat 9 or newer version)
